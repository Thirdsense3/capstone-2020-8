# 임산부를 위한 더 나은 좌석 (가명)


### 1. 프로젝트 소개


부산시에서 현재 운영하고 있는 임산부 표시는 보급률이 낮고, 임신 초기에는 겉으로도 임산부임이 표시가 잘 안나기 때문에 양보받기가 힘들다. 
그래서 좀 더 임산부들의 편의를 위해서, 현재 모든 사람들에게 보급되어진 스마트폰의 어플을 이용해, 임산부들이 좌석을 좀 더 편하게 앉고 내릴 수 있도록 어플과 좌석을 발전시키도록 했다.
